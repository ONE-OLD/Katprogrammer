Learn HTML


Welcome to HTML learn page, htis site will help you to understand HTML and all tags used in HTML. At the end of this lesson, you will be able to make sample website. These tags will help you to complete your website and the function of its tag.

HTML in full: Hyper Text Markup Language

  • Basic stucture:

    • <!DOCTYPE html>
    • <html>
    • <head>
    • <title>website name</title>
    • </head>
    • <body>

    • Content goes here

    • </body>
    • </html>

    • Headings :

    • <h1> Heading 1 </h1> - most important
    • <h2> Heading 2 </h2>
    • <h3> Heading 3 </h3>
    • <h4> Heading 4 </h4>
    • <h5> Heading 5 </h5>
    • <h6> Heading 6 </h6> - least important

    • Paragraph:

    • <p> Your paragraph goes here. </p>

    • Link:

    • <a href="http://www.example.com"> Link text. </a>
    • <a href="example.html"> Link text. </a>

    • Image:

    • <img src="example.png" altitude="">

    • Video:

    • <video src="example.mp4" controls>

    • Unordered list:

    • <ul>
    • <li>item 1</li>
    • <li>item 2</li>
    • </ul>

    • Ordered list:

    • <ol>
    • <li>item 1</li>
    • <li>item 2</li>
    • </ol>

    • Table:

    • <table border="4">
    • <tr> - table row
    • <th>Header 1</th> - table head
    • <th>Header 2</th>
    • </tr>
    • <tr>
    • <td>Data 1</td> - table data
    • <td>Data 2</td>
    • </tr>
    • </table>

    • Semantic Elements:

    • <header> Header </header>
    • <nav> Navigation </nav>
    • <section> Section </section>
    • <article> Article </article>
    • <aside> Aside </aside>
    • <footer> Footer </footer>
    • <div> Content. </div>
    • <script src="example.js"> Connect to javascript. </script>
    • <progerss> Download. </progress>

    • Audio:

    • <audio src="example.mp3" controls><audio>

    • Html formatting:

    • <b> - Bold text
    • <i> - Italic text
    • <mark> - Highlighted text
    • <del> - Deleted text crossline
    • <sup> - superscript | top
    • <sub> - Subscript | bottom

    • HTML input types :

    • <input type="text">
    • <input type="password">
    • <input type="number">
    • <input type="email">
    • <input type="url">
    • <input type="date">
    • <input type="time">
    • <input type="datetime-local">
    • <input type="month">
    • <input type="week">
    • <input type="color">
    • <input type="checkbox">
    • <input type="radio">
    • <input type="file">
    • <input type="submit">
    • <input type="reset">
    • <input type="button">
    • <input type="hidden">
    • <input type="range">
    • <input type="search">

    • Links in Head Tag:

    • <link rel="stylesheet" href="example.css">
    • <link rel="icon" href="example.PNG" type="image/x-icon">
    • <link rel="shortcut icon" type="image/x-icon" href="/favicon.ico">
    • <link rel="import" href="example.html">
    • <link rel="alternate" href="example.xml" type="application/atom+xml" title="example">
    • <link rel="prev" href="http://stackoverflow.com/documentation/java/topics">
    • <link rel="next" href="http://stackoverflow.com/documentation/css/topics">
    • <link rel="alternate" type="application/atom+xml" href="http://example.com/feed.xml">
    • <link rel="preconnect" href="URL">
    • <link rel="dns-prefetch" href="URL">
    • <link rel="prefetch" href="URL">
    • <link rel="prerender" href="URL">
    • <link rel="alternate stylesheet" href="path/to/style.css" title="yourTitle">

    • Comment:

    • <!-- comment or discusion -->
